Save Directory: The save directory for incoming files.
Legacy Mode for Hotspot: Use 'netsh' method to create a hotspot instead. (Slower and less compatibility with many PCs.)
Hotspot SSID: The SSID to create a WiFi Hotspot with.
(You must change the SSID in shortcuts.)
Hotspot Password: The password to create a WiFi Hotspot with.
Auto Update: Check for updates on startup and update if found. (Recommended as the app might not be very stable.)
Port Number: (Advanced) If you change this, you will have to mess with the shortcuts too.
